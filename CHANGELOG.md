# Changelog

Все значимые изменения в проекте DomoHub документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [0.1.0] - 2026-07-20

### Добавлено

- **Backend API** на FastAPI с REST и WebSocket
- **Плагинная архитектура** для интеграции устройств разных производителей
- Плагины: Demo, HTTP/REST, MQTT, Tuya (Smart Life)
- **Web UI** — React + Tailwind, тёмная тема, адаптивный дизайн
- Управление устройствами: свет, розетки, датчики, климат, шторы
- Комнаты и группировка устройств
- **n8n / LLM интеграция** — webhook API для голосового и текстового управления
- **Docker Compose** для развёртывания на сервере
- MQTT брокер (Mosquitto) как опциональный сервис
- Документация: архитектура, деплой, интеграция с n8n
- Демо-устройства для тестирования без реального железа
